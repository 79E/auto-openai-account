# UI Style

The chosen direction is `docs/ui-previews/02-modern-saas.html`.

## Name

Modern SaaS Console.

## Visual Rules

- Light background.
- Subtle blue and purple gradients.
- Rounded cards.
- Clean navigation.
- Dashboard uses larger cards and calm spacing.
- Tables and log views can be denser where workflow efficiency matters.

## React Implementation Notes

- Use React + TypeScript.
- Use Tailwind CSS.
- Use shadcn/ui-style primitives: Card, Button, Badge, Table, Tabs, Dialog, Sheet, Toast.
- Avoid old enterprise admin visuals.
- Keep realtime logs visible and readable.
